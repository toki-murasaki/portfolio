【画像ファイルの配置方法】

このフォルダに以下のファイルを配置してください。

■ プロフィール画像
  profile.jpg  — About セクションに表示されます

■ オリジナルキャラクター画像
  haiiro.jpg   — 灰色ちゃんセクションに表示されます

■ ギャラリー作品
  work01.jpg
  work02.jpg
  work03.jpg
  work04.jpg
  work05.jpg
  work06.jpg
  … （work07.jpg 以降を追加する場合は index.html の
      galleryImages 配列にエントリを追記してください）

■ 推奨サイズ
  プロフィール : 600×800px 以上 (縦長)
  キャラクター : 800×800px 以上 (正方形〜縦長)
  ギャラリー   : 600×750px 以上 (縦長 4:5 比率)

  ファイル形式は JPG / PNG / WebP いずれも対応しています。
